# PlayZone React

This repository contains a React conversion of the PlayZone sports booking portal.

## Run locally

1. Install packages:

```bash
npm install
```

2. Start the dev server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.

## App routes

- `/` — Home page
- `/booking` — Slot booking form
- `/info` — Terms and classes
- `/sports/cricket` — Cricket details
- `/sports/football` — Football details
- `/sports/badminton` — Badminton details
